# tamp themes

> Three theme families, dark and day. Part of the [tamp](https://github.com/tamp-workshop) toolkit.

Available for VSCode and Vim.

---

## Themes

| Family | Dark | Day |
|--------|------|-----|
| **Lunar Lobby** | `lunar-lobby` | `lunar-lobby-day` |
| **Space Rumours** | `space-rumours` | `space-rumours-day` |
| **Vanguard Outpost** | `vanguard-outpost` | `vanguard-outpost-day` |

---

## VSCode

**Via Marketplace** *(coming soon)*

**Manual install:**

```sh
git clone https://github.com/tamp-workshop/themes.git
cp -r themes/vscode ~/.vscode/extensions/tamp-themes
```

Restart VSCode, then select a theme via `⌘K ⌘T`.
